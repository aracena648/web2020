HTML basico

HTML (Lenguaje de Marcas de Hipertexto, o Hypertext Markup Language en inglés) es el lenguaje usado para crear documentos web. ... Cuando se usa como extensión de archivo, ésta le informa a un explorador web como Firefox que el archivo es un documento web.
